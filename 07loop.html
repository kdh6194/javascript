javascript 반복문

while

조건식이 참일 동안 코드블록을 실행하는 반복문

만일, 조건식이 처음부터 거짓이면 코드블록은 한번도 실행되지 않을 수 있음

for는 반복실행 횟수를 처음부터 알고 있을때 실행하는 반복문인 반면, while은 반복 횟수를 모르고 실행하는 반복문임

하지만, for문을 while문으로 또는 while문을 for문으로 변환이 가능

while 구조

초기식

while문(조건식) {

실행할 코드들

증감식

}
